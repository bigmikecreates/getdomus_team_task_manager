## Understanding the Dashboard

The dashboard is your home screen. It shows:

- Task statistics (total, by status, by priority)
- Overdue and critical tasks requiring attention
- Upcoming deadlines
- Recent activity

See the [Dashboard guide](dashboard.md) for a detailed walkthrough.

## Key Concepts

### Tasks

Tasks represent work items. Each task has:

- **Title** and optional **description**
- **Status:** To Do → In Progress → In Review → Done
- **Priority:** Low, Medium, High, Critical
- **Due date** (optional)
- **Assigned developers** (one or more)

### Developers

Developers are team members. Each developer profile includes:

- Name and email
- Timezone (for distributed teams)
- Working hours (optional, used for availability display)

### Assignments

Tasks can be assigned to multiple developers. This lets you track who is working on what across distributed teams.
