# Dashboard

The dashboard is your home screen when you log in. It gives you a quick overview of the project.

## Overview

<!-- SCREENSHOT: dashboard-stats-cards
     What to capture: Stats cards section showing task counts by status and priority
     Suggested size: 1280x400
-->

The top section shows key metrics:

- **Total Tasks** — Overall task count
- **By Status** — Breakdown: To Do, In Progress, In Review, Done
- **By Priority** — Breakdown: Low, Medium, High, Critical

## Attention Required

<!-- SCREENSHOT: dashboard-attention
     What to capture: Attention Required section showing overdue and critical tasks
     Suggested size: 1280x400
-->

This section highlights tasks that need immediate focus:

- **Overdue Tasks** — Tasks past their due date
- **Critical Tasks** — Tasks marked as critical priority

Click any task in this list to go directly to its detail view.

## Upcoming Tasks

<!-- SCREENSHOT: dashboard-upcoming

## Sidebar Navigation

<!-- SCREENSHOT: sidebar-navigation
     What to capture: Sidebar showing available navigation links
     Suggested size: 300x800
-->

The sidebar provides quick access to:

| Link | Available To | Description |
|------|-------------|-------------|
| **Dashboard** | All | This overview page |
| **Tasks** | All | Task list and management |
| **Developers** | All | Team member list with availability |
| **New Developer** | Admin only | Add a team member |

On mobile, tap the hamburger menu icon to open the sidebar.
